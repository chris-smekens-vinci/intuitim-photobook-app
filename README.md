# intuitim-photobook-app
PhotoBook learning project built during an internship at Intuitim, using NestJS for the backend and Angular for the frontend. Focus on clean architecture and best practices.

## Tech stack
- Backend: NestJS
- Frontend: Angular

## Architecture
- Monorepo
- Modular monolith backend
- SPA frontend

## Goal
Learn NestJS and Angular from scratch with clean enterprise practices.
